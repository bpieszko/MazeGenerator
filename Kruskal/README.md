# MazeGenerator - using Kruskal's algorithm

## Build
```
git clone https://github.com/bpieszko/MazeGenerator.git
cd MazeGenerator/Kruskal
mkdir build
cd build
cmake ..
make
```

## Using
```
./MazeGenerator [size]
```
example:
```
./MazeGenerator 100
```
