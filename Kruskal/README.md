# MazeGenerator - using Kruskal's algorithm

## Build
```
git clone https://github.com/bpieszko/MazeGenerator.git
cd MazeGenerator
mkdir build
cd build
cmake ..
make
```

## Using
```
./MazeGenerator [height] [width]
```
